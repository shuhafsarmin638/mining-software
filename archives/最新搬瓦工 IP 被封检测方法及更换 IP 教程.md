输入搬瓦工 VPS 的 IP 地址和端口，便可查看检测结果。结果通常分为以下几类：

- **TCP 可用**：表示国内或国外的连接正常。

通过这些信息，您可以判断 IP 的具体状况。

- **IP NOT BLOCKED**：表示 IP 正常，无需更换。
